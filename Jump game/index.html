Jump game

Instructions:

Given a random generated array in which index starts from 0 instead of 1. By adding current element's value to current element's index to check whether there is a way to jump from start to the end.

For example, given 2 4 1 3 0 and start from 2 whose index is 0 and value is 2. There are three ways to jump to the end:

2->1->3 or 2->4 or 2->4->1->3.

Rules:

There are three different levels: easy level has 5 numbers, noraml level has 10 numbers, hard level has 15 numbers.

Correct anwser will get 5 points, wrong answer will lose 2 points

User can click "Play again" button to play next game and click "Reset game" button to start over.
